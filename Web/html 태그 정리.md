## 테이블 고정

style : table-layout: fixed;

## html 색상표

https://wepplication.github.io/tools/colorPicker/

## 가운데 정렬

<center></center>

센터 태그

## 진하게, 강조 표시

<b></b>

b 태그

## 글자색 변경
